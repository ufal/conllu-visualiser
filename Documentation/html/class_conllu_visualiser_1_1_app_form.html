An event which occurs after MouseDown event, where the MovedPoint is chosen. If possible, the word, on whose point the MouseUp event occured, is the new parent of the moved word.

Any of words can not be null and the parent word must be able to be the parent of the new word. (The parent is th descendant of the child, the parent is a special word, which can not have any children...)

◆ VisualiseNewGraph()

void ConlluVisualiser.AppForm.VisualiseNewGraph ( )

A method to visualise the new graph of the active sentence, used if the graph is changed.

Change can be in the sentence (different sentence, change in structure,..) or in the size of the form. If there is no active sentence(it actually means that there is no sentence), the graph space is empty.
